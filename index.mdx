WinUI is the modern UI layer for Windows application development, providing a rich set of controls and styles that implement the Fluent Design System. Whether you're building a new Windows app or modernizing an existing one, WinUI 3 — which ships as part of the [Windows App SDK](https://learn.microsoft.com/windows/apps/windows-app-sdk/) — gives you full access to the latest Windows UI patterns, animations, and accessibility features.

## Why WinUI?

WinUI is the recommended UI framework for new Windows application development. It combines the power of native Win32 performance with modern Fluent Design visuals and Windows Runtime APIs.

<CardGroup cols={2}>
  <Card title="Fluent Design built in" icon="sparkles">
    Every control ships with Fluent Design styles, animations, and light/dark theme support out of the box.
  </Card>
  <Card title="C# and C++ support" icon="file-code">
    Write apps in C# with .NET or C++ with WinRT. WinUI works with both language projections.
  </Card>
  <Card title="Accessible by default" icon="universal-access">
    All controls include AutomationPeer implementations for screen readers and assistive technologies.
  </Card>
  <Card title="Windows App SDK" icon="windows">
    Ships as part of the Windows App SDK, targeting Windows 10 1809 (Build 17763) and newer.
  </Card>

## Get started

<Steps>
  <Step title="Install Visual Studio and the Windows App SDK">
    Install Visual Studio 2022 with the **Windows application development** workload, which includes the Windows App SDK and WinUI 3 templates.
  </Step>
  <Step title="Create a new WinUI 3 project">
    In Visual Studio, create a new project using the **Blank App, Packaged (WinUI 3 in Desktop)** template.
  </Step>
  <Step title="Add controls to your XAML">
    Reference WinUI controls in your XAML pages using the `Microsoft.UI.Xaml.Controls` namespace.
  </Step>
  <Step title="Run the WinUI 3 Gallery">
    Install the [WinUI 3 Gallery](https://apps.microsoft.com/detail/9P3JFPWWDZRC) from the Microsoft Store to see every control in action with live, editable XAML samples.
  </Step>
</Steps>

<Note>
  Install the WinUI 3 Gallery from the Microsoft Store to explore all controls with live, editable examples before writing any code.
</Note>
